# terminal-fractal-viewer

Depencencies: ncurses

Installation with apt:
\n sudo apt get-install libncurses-dev libncursesw6

Google for other package managers.

This might work if you can figure out how to install PDCurses for windows: https://pdcurses.org/

You'll also need to have gcc or another compiler obviously.
I think thats it though. 

CONTROLS:

WASD to move around
'b' to zoom in
'v' to zoom out
